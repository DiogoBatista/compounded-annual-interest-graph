# Compounded Interests with Monthly Contributions

This is an example how to use React w/ Typescript and ChartJs to build a graph where you can visualize your investments along the time.

## Demo
- https://compounded-interests-graph.appspot.com/

Get started:
- Download or clone the repo
- run `npm install`
- run `npm start`
